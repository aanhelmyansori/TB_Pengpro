# Sistem Manajemen Inventaris Gudang (CLI)

Aplikasi CLI untuk simulasi manajemen inventaris gudang skala kecil — tugas besar mata kuliah Pemrograman (Telkom University).

## Fitur
- ✅ Tambah, lihat, update, hapus barang  
- ✅ Validasi kode unik & stok non-negatif  
- ✅ Laporan ringkasan (total stok, stok kritis)  

## Cara Menjalankan
```bash
go run main.go
